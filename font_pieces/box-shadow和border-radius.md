##border-radius
	画实心圆的方法：宽度和高度一致(正方形)，然后四个角设置为高度或者宽度的1/2.
	
	```
	<div class="circle"></div>
	.circle{
	    width:100px;
	    height:100px;
	    background-color:#cb18f8;
	    border-radius:50px;
	}
	```
